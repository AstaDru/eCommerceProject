Welcome to our E-commerce Shop api

ROUTES

Browsing

GET - /api/browse

=> Returns an Array of all items in shop as objects

GET - /api/browse?{name}

=> Returns an item object by {name} which is passed as a url param

{ id: sting, name: string, price: integer, quantity: integer, description: string }

User

POST - /api/register

=> Creates a new user account and returns the user object {name: string, surname: string, email: string, password: string, address: string }

POST - /api/login

=> authenticates the user by fetching account using email {email: a@b.c, password: secret123 }

PUT - /api/settings ~ authenticating required

=> takes in to values {attribute: string, value: string } where attribute represents the users attribute key and value as the new value.

DELETE - /api/deleteuser ~ authenticating required

=> Deletes user using id of current session user

=> # This action deletes all records of transactions made by user (the missing quantity of items is the only indicator of transaction at current)

Cart

GET - /api/cart ~ authenticating required

=> Returns an Array of all items in cart as objects

POST - /api/cart/additem ~ authenticating required

=> adds an item from items to cart { itemName: string, quantity: integer }

PUT - /api/cart/changeqty ~ authenticating required

=> Set new quantity of item in cart { itemName: string, quantity: integer }

DELETE - /api/cart/removeitem ~ authenticating required

=> remove an item from cart {itemName: string }

DELETE - /api/cart/clear ~ authenticating required

=> Removes all items inside the cart

GET - /api/cart/checkout ~ authenticating required

=> makes an order with all current cart items, returning the order id

orders

GET - /api/orders ~ authenticating required

=> Returns an Array of all orders made by user
