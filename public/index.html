Welcome to our E-commerce Shop api

ROUTES

  • /api/browse - GET => list all items in shop
  • /api/register - POST => create user account
  • /api/login - POST => Login in user
  • /api/add_to_cart - PUT (authenticated) => add item to cart using item_id
  • /api/remove_from_cart - PUT (authenticated) => remove item from cart using item_id
  • /api/checkout - POST (authenticated) => Push cart_items to order
  • /api/order - GET (authenticated) => view all orders
  • /api/settings - PUT (authenticated) => Change user attributes
