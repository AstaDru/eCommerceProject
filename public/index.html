Welcome to our E-commerce Shop api

ROUTES

Browsing

GET - /api/browse

=> Returns an Array of all items in shop as objects

GET - /api/browse?{name}

=> Returns an item object by {name} which is passed as a url param

{ id: sting, name: string, price: integer, quantity: integer, description: string }

User

POST - /api/login

=> authenticates the user by fetching account using email {email: a@b.c, password: secret123 }

POST - /api/register ~ authenticating required

=> Creates a new user account and returns the user object

PUT - /api/settings ~ authenticating required

=> takes in to values {attribute: password, value: secret1234 } where attribute represents the users attribute key and value as the new value.

DELETE - /api/deleteuser ~ authenticating required

=> Deletes user using id of current session user

{ id: sting, name: string, surname: string, email: string, password: string, address_id: integer }

Cart

GET - /api/cart ~ authenticating required

=> Returns an Array of all items in cart as objects

PUT - /api/addtocart ~ authenticating required

=> adds an item from items to cart {itemName: string, cartId: string, quantity: integer }

DELETE - /api/removefromcart ~ authenticating required

=> remove an item from items to cart {itemName: string, cartId: string }

{ id: sting, cart_id: string, item_id: integer, quantity: integer }
  • /api/checkout - POST (authenticated) => Push cart_items to order
  • /api/order - GET (authenticated) => view all orders
